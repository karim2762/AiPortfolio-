# ✨ Kexer Portfolio

A modern, elegant, and responsive developer portfolio built to showcase my projects, skills, and journey as a Full Stack Developer.

Designed with a clean aesthetic inspired by modern interfaces, featuring smooth animations, glassmorphism effects, dark/light themes, and an exceptional user experience across all devices.

🌐 **Live Preview:** https://kexer-port.netlify.app/

- 🎨 Modern and professional UI design

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/karim2762/AiPortfolio-.git
```

### Open locally

Simply open the `index.html` file in your browser.

Or use a local server:

```bash
python -m http.server
```

Then visit:

```text
http://localhost:8000
```
